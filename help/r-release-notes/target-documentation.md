# Target 설명서 개요

[!DNL Adobe Target] 설명서는  [!DNL Target] UI에 대한 가이드, 자습서, 비디오, 웨비나, API 및 SDK를 비롯한 다양한 형식으로 제공됩니다.

다음은 [!DNL Target]에 사용할 수 있는 가장 일반적인 문서 유형에 대한 간략한 설명입니다.
