# [!DNL Target]을 [!DNL Adobe Experience Manager] (AEM)와 통합

자세한 내용은 [AEM 경험 조각 및 콘텐츠 조각 개요](/help/main/c-integrating-target-with-mac/aem/aem-experience-and-content-fragments.md)를 참조하십시오.
