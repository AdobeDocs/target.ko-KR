# 구현하기 전에

구현하기 전에 고려해야 할 사항입니다 [!DNL Adobe Target].

이 섹션에는 개발자가 수집해야 하는 구현, 개인 정보 고려 사항, 쿠키 관련 설명, Target으로 데이터를 가져오는 다양한 방법 목록, 보안 고려 사항 및 TLS(전송 계층 보안) 암호화 관련 정보가 포함되어 있습니다.
