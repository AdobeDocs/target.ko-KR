* 3개의 장바구니 기반 알고리즘에서 [!DNL Target] 백엔드. (TGT-43456)
* 활성화됨 [!DNL Target] 다음을 사용하여 활성화된 조직의 UI 토큰 새로 고침 [비즈니스 ID 계정](https://helpx.adobe.com/enterprise/using/identity.html){target=_blank} 및 PBA(정책 기반 인증). (TGT-42590)
