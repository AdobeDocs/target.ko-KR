이 릴리스에는 다음과 같은 새로운 기능이 포함되어 있습니다.

| 기능 | 세부 사항 |
|--- |--- |
| Headless 개인화 및 실험을 위한 AEM 콘텐츠 조각 | [!DNL Target] 활동에서 [!DNL Adobe Experience Manager] (AEM) [!UICONTROL 콘텐츠 조각]을 사용합니다. AEM의 편의성과 기능을 강력한 AI(인공 지능) 및 ML(기계 학습) 기능과 결합할 수 있습니다. [!DNL Target] 헤드리스 개인화 및 실험을 지원하기 위해 입니다. |
