# [!UICONTROL 시각적 경험 작성기] 변경 사항

[!DNL Adobe Target Standard/Premium] 25.2.1 릴리스(2015년 2월 17일)에서는 업데이트된 [!UICONTROL 시각적 경험 작성기]&#x200B;(VEC)가 도입되었습니다. 이 문서에서는 VEC의 이전 버전과 업데이트된 버전의 차이점에 대해 설명합니다.

>[!IMPORTANT]
>
>업데이트된 [!UICONTROL 시각적 편집 작성기]에는 [!DNL Chrome Web Store]에서 사용할 수 있는 [!DNL Adobe Experience Cloud] [[!UICONTROL 시각적 편집 도우미] 확장](/help/main/c-experiences/c-visual-experience-composer/r-troubleshoot-composer/visual-editing-helper-extension.md)이 필요합니다.

VEC는 기존 활동을 작성하거나 편집할 때 표시됩니다.

![VEC(시각적 경험 작성기)](/help/main/c-experiences/c-visual-experience-composer/assets/vec-highlight-refresh.png)

## VEC에 대한 주요 변경 사항

다음 섹션에서는 이전 버전과 비교하여 업데이트된 VEC의 주요 변경 사항에 대해 설명합니다.

### [!UICONTROL 경험] 레일

이전 버전과 마찬가지로 [!UICONTROL 경험] 레일은 VEC 왼쪽에 남아 있습니다. [!UICONTROL 경험] 레일을 축소할 수 없습니다.

![경험 레일](/help/main/c-experiences/c-visual-experience-composer/assets/experiences-panel.png)

[!UICONTROL 경험] 레일을 사용하여 경험을 만들거나 이름을 바꾸거나 제거할 수 있습니다. 새 경험을 추가하려면 **[!UICONTROL 추가]** 아이콘( ![추가 아이콘](/help/main/assets/icons/Add.svg))을 클릭하십시오. 경험을 복제, 삭제 또는 리디렉션하려면 [!UICONTROL 추가 작업] 아이콘(![추가 작업 아이콘](/help/main/assets/icons/MoreSmall.svg))을 클릭하십시오.

### [!UICONTROL 구성 요소] 레일(신규)

새 [!UICONTROL 구성 요소] 레일을 사용하여 웹 페이지에 여러 구성 요소를 추가하고 필요에 따라 편집할 수 있습니다.

![구성 요소 레일](/help/main/c-experiences/c-visual-experience-composer/assets/components-panel.png)

새 구성 요소를 추가하려면 [!UICONTROL 구성 요소] 레일에서 원하는 구성 요소를 선택하고 페이지의 기존 요소를 마우스로 가리킨 다음 선택한 요소 앞의 구성 요소를 삽입하도록 선택합니다.

>[!NOTE]
>
>이 영역에 [!UICONTROL 구성 요소] 레일 대신 [!UICONTROL 수정 사항] 레일이 표시되면 **[!UICONTROL 구성 요소 표시]** 아이콘(![구성 요소 표시 아이콘](/help/main/assets/icons/Add.svg))을 클릭합니다. [!UICONTROL 구성 요소 표시] 아이콘( ![구성 요소 표시 아이콘](/help/main/assets/icons/Add.svg))과 [!UICONTROL 수정 사항 표시] 아이콘( ![수정 사항 표시 레일](/help/main/assets/icons/History.svg))은 적절한 옵션을 표시하도록 전환합니다.
>
>[!UICONTROL 구성 요소] 레일이 열려 있는 동안 [!UICONTROL 구성 요소] 레일을 축소하고 [!UICONTROL 디자인] 캔버스를 확대하려면 (![구성 요소 표시 아이콘](/help/main/assets/icons/Add.svg)) 아이콘을 클릭합니다.

### [!UICONTROL 수정] 레일

[!UICONTROL 수정 사항] 레일을 열려면 [!UICONTROL 구성 요소] 레일에서 [!UICONTROL 수정 사항 표시] 아이콘(![수정 사항 표시 레일](/help/main/assets/icons/History.svg))을 클릭합니다. [!UICONTROL 수정 사항] 레일이 편집 캔버스의 오른쪽에서 왼쪽으로 위치가 변경되었습니다.

![수정 레일](/help/main/c-experiences/c-visual-experience-composer/assets/modifications-panel.png)

>[!NOTE]
>
>[!UICONTROL 구성 요소 표시] 아이콘( ![구성 요소 표시 아이콘](/help/main/assets/icons/Add.svg))과 [!UICONTROL 수정 사항 표시] 아이콘( ![수정 사항 표시 레일](/help/main/assets/icons/History.svg))은 적절한 옵션을 표시하도록 전환합니다.
>
>[!UICONTROL 수정 사항] 레일이 열려 있는 동안 [!UICONTROL 수정 사항] 레일을 축소하고 [!UICONTROL 디자인] 캔버스를 확대하려면 [!UICONTROL 수정 사항 표시] 아이콘(![수정 사항 레일 표시](/help/main/assets/icons/History.svg) ) 아이콘을 클릭합니다.

[!UICONTROL 수정 사항] 레일에는 VEC에서 페이지에 적용된 모든 변경 사항이 표시되며 추가 변경을 수행할 수 있습니다(예: CSS 선택기, Mbox, 사용자 지정 코드).

수정 사항을 추가하거나, 모든 수정 사항을 삭제하거나, 모든 잘못된 수정 사항을 삭제하려면 [!UICONTROL 추가 옵션] 아이콘(![추가 작업 아이콘](/help/main/assets/icons/MoreSmall.svg))을 클릭하십시오. 일괄 작업을 수행하려면 [!UICONTROL 선택]을 클릭하세요. [!UICONTROL 모든 페이지에 적용] 또는 [!UICONTROL 삭제].

[!UICONTROL 수정 사항] 레일을 다시 표시하려면 [!UICONTROL 수정 사항] 레일에서 [!UICONTROL 수정 사항 숨기기] 아이콘(![수정 사항 레일 표시](/help/main/assets/icons/History.svg))을 클릭하십시오.

### [!UICONTROL 속성] 레일(신규)

[!UICONTROL 속성] 레일을 사용하면 해당 요소가 HTML 요소인지 [!DNL Target]과(와) 관련된 개체인지 여부와 관계없이 페이지에서 선택한 요소의 속성을 변경할 수 있습니다(예: 권장 사항 또는 오퍼).

![속성 레일](/help/main/c-experiences/c-visual-experience-composer/assets/properties-panel.png)

레일 상단의 아이콘을 클릭하여 HTML 코드를 편집하거나 요소를 삭제, 복제 또는 숨깁니다. 변경 사항이 [!UICONTROL 수정 사항] 레일에 표시됩니다.

![속성 아이콘](/help/main/c-experiences/c-visual-experience-composer/assets/options-icons.png)

[!UICONTROL 속성] 레일은 오른쪽 레일에서 접을 수 있습니다. [!UICONTROL 속성] 레일을 축소하거나 표시하려면 레일의 오른쪽에 있는 [!UICONTROL 속성 표시/숨기기] 아이콘(![속성 아이콘](/help/main/assets/icons/Propertie.svg))을 클릭하십시오.

### 활동 설정/구성

디자인 캔버스 위에 표시된 [!UICONTROL 구성] 아이콘(![구성 아이콘](/help/main/assets/icons/Setting.svg))을 클릭하여 활동 속성 메뉴를 표시합니다.

![활동 구성 옵션](/help/main/c-experiences/c-visual-experience-composer/assets/configure-options.png)

다양한 옵션을 사용하여 속성을 지정하고, 페이지 전달 규칙을 편집하고, 사이트 환경 설정을 지정하고, 페이지를 추가로 추가하고, 다중 페이지 또는 여러 대상 활동을 활성화 또는 비활성화할 수 있습니다. [!UICONTROL 속성] 할당은 [[!DNL Target Premium]](/help/main/c-intro/intro.md#premium) 기능입니다.

위치 및 기능은 이전 VEC UI와 유사합니다.

### [!UICONTROL 디자인]/[!UICONTROL 찾아보기] 모드

[!UICONTROL 속성] 레일의 맨 위에 표시되는 [!UICONTROL 디자인]/[!UICONTROL 찾아보기] 전환을 사용하여 디자인과 찾아보기 모드를 전환하십시오.

![디자인 및 찾아보기 전환](/help/main/c-experiences/c-visual-experience-composer/assets/design-browse-mode.png)

[!UICONTROL 찾아보기] 모드를 사용하여 사이트를 탐색하고 업데이트할 보기 또는 페이지를 선택하십시오. 변경 내용을 추가하거나 편집하려면 [!UICONTROL 디자인] 모드로 다시 전환하십시오.

### [!UICONTROL 실행 취소]/[!UICONTROL 다시 실행]

[!UICONTROL 실행 취소] 아이콘(![실행 취소 아이콘](/help/main/assets/icons/Undo.svg))을 클릭하여 변경 내용을 실행 취소할 수 있습니다.

![VEC의 실행 취소 아이콘](/help/main/c-experiences/c-visual-experience-composer/assets/undo.png)

작업을 다시 실행하려면 실행 취소/[!UICONTROL 다시 실행] 단추 그룹을 확장하고 [!UICONTROL 다시 실행]을 선택하세요.

### [!UICONTROL 디자인] 캔버스

[!UICONTROL 디자인] 캔버스를 사용하면 화면 맞춤, [!UICONTROL 데스크톱], [!UICONTROL 태블릿], [!UICONTROL 모바일 가로] 및 [!UICONTROL 모바일 세로]를 포함한 뷰포트를 선택할 수 있습니다.

![뷰포트 옵션](/help/main/c-experiences/c-visual-experience-composer/assets/viewports.png)

또한 업데이트된 VEC를 사용하면 적절한 아이콘(![확대 아이콘](/help/main/assets/icons/ZoomIn.svg) 또는 ![축소 아이콘](/help/main/assets/icons/ZoomOut.svg))을 클릭하여 확대하거나 축소할 수 있습니다.
