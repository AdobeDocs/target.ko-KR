# 콘텐츠 검색 및 필터링

[!DNL Adobe Target]의 [!UICONTROL 오퍼] 라이브러리에서 키워드로 자산을 검색합니다.

## 키워드로 코드 오퍼 검색

1. **[!UICONTROL 오퍼]** > **[!UICONTROL 코드 오퍼]**&#x200B;를 클릭합니다.
1. **[!UICONTROL 검색 위치]** 상자에 원하는 키워드를 입력하십시오.

   드롭다운을 사용하여 [!UICONTROL 오퍼 이름], [!UICONTROL AEM 경로] 또는 [!UICONTROL AEM 태그]별로 필터링할 수 있습니다.

   [!UICONTROL 필터 표시] 아이콘 ![필터 표시 아이콘](/help/main/assets/icons/Filter.svg)을 클릭하여 [!UICONTROL Type], [!UICONTROL Source] 및 [!UICONTROL AEM Type] 옵션을 사용하여 검색을 세분화할 수도 있습니다.

## 키워드로 이미지 오퍼 검색 {#section_2465A71BC95942588F586B1EC8B9E5DB}

1. **[!UICONTROL 오퍼]** > **[!UICONTROL 이미지 오퍼]**&#x200B;를 클릭합니다.

1. (선택 사항) [!UICONTROL 카드 보기]와 [!UICONTROL 목록 보기] 사이를 전환하고 [!UICONTROL 자산] 라이브러리의 오른쪽 위 모서리에 있는 [!UICONTROL 카드 보기] 아이콘(![카드 보기 아이콘](/help/main/assets/icons/ViewCard.svg) ) 또는 [!UICONTROL 목록 보기] 아이콘(![목록 보기 아이콘](/help/main/assets/icons/ViewList.svg) )을 클릭합니다.
1. 왼쪽 상단 모서리에 있는 **[!UICONTROL 컨텐츠 전용]** 아이콘(![컨텐츠 전용 아이콘](/help/main/assets/icons/RailLeft.svg) )을 클릭하여 검색 상자를 표시합니다.
1. 검색 상자에 찾을 자산의 키워드를 입력한 다음 [!UICONTROL Enter]를 누릅니다.
1. (조건부) 검색을 구체화하려면 왼쪽의 **[!UICONTROL 필터 및 옵션]** 창을 사용하십시오.

   다음을 기준으로 필터링할 수 있습니다.

   * **[!UICONTROL 파일 형식]**:
      * [!UICONTROL 이미지]
      * [!UICONTROL 문서]
      * [!UICONTROL 멀티미디어]
      * [!UICONTROL 보관]
   * **[!UICONTROL 파일 크기]**: 슬라이더를 사용하여 원하는 파일 크기를 선택하십시오. [!UICONTROL 가장 작게], [!UICONTROL 작게], [!UICONTROL Medium], [!UICONTROL 크게] 또는 [!UICONTROL 가장 크게].
   * **[!UICONTROL 마지막 수정일]**: 슬라이더를 사용하여 기간을 선택하십시오. [!UICONTROL 최근], [!UICONTROL 시간], [!UICONTROL 일], [!UICONTROL 주], [!UICONTROL 월], [!UICONTROL 년] 또는 [!UICONTROL 모든 Assets].
   * **[!UICONTROL 승인 상태]**: [!UICONTROL 승인됨] 또는 [!UICONTROL 거부됨]
   * **[!UICONTROL 만료 상태]**: [!UICONTROL 만료됨].
   * **[!UICONTROL 평균 등급]**: 1에서 5.
   * **[!UICONTROL 방향]**: [!UICONTROL 가로], [!UICONTROL 세로] 또는 [!UICONTROL 사각형].
   * **[!UICONTROL 스타일]**: [!UICONTROL 색상] 또는 [!UICONTROL 흑백].
   * **[!UICONTROL 사용 점수]**: [!UICONTROL 최소]에서 [!UICONTROL 최대]까지
   * **[!UICONTROL 노출 점수]**: [!UICONTROL 최소]에서 [!UICONTROL 최대]까지
   * **[!UICONTROL 점수를 클릭하세요]**: [!UICONTROL 최소]에서 [!UICONTROL 최대].
