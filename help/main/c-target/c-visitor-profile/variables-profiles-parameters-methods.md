## 고객 속성 {#section_62B4821EB6564FF4A14159A837AD4EDB}

고객 속성이 `crs.get('<Datasource Name>.<Attribute name>')`.

이러한 속성이 프로필 스크립트에서 토큰으로 사용되거나 프로필 스크립트가 없어도 오퍼에서 직접 토큰으로 사용될 수도 있습니다. 토큰은 다음 형식이어야 합니다. `${crs.datasourceName.attributeName}`. 의 공백은 `datasourceName` 는 모든 API 호출에서 제거됩니다.
