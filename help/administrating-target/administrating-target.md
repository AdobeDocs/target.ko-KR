# Target 관리{#administer-target}

>[!NOTE]
>
>아래 나와 있는 이 섹션의 주제는 [!DNL Target Standard/Premium] 20.6.1 릴리스(2020년 7월)에 제공되는 UI 변경 사항에 대한 간략한 개요를 제공하기 위해 업데이트되었습니다. 이러한 항목에 제시된 대부분의 정보는 현재 UI에 적용됩니다. 그러나 옵션이 약간 다를 수 있습니다.
