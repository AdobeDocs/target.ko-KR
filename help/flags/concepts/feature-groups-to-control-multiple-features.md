# 여러 기능을 제어하는 기능 그룹 {#feature-groups}

[기능 플래그](what-is-a-feature-flag.md)는 단일 기능을 제어합니다. 여러 관련 기능 플래그를 함께 관리하고 동일한 대상에 도달하는지 확인해야 하는 경우 **기능 그룹**&#x200B;을 사용합니다.

## 기능 그룹이 수행하는 작업 {#what-it-does}

기능 그룹은 여러 기능 플래그를 번들로 제공하고 단일 대상 규칙을 모든 대상에 한 번에 적용할 수 있도록 합니다. 이 기능은 단일 사용자 대면 기능이 여러 애플리케이션 또는 서비스에 걸쳐 있을 때 유용합니다.

예를 들어 데스크탑 애플리케이션, 모바일 앱 및 백엔드 서비스 간의 변경 사항을 포함하는 공동 작업 기능을 생각해 보십시오. 세 가지 응용 프로그램 모두에서 플래그가 있는 기능 그룹을 생성하면 사용자의 1%에게 해당 기능을 열 수 있으며, 이러한 사용자가 세 표면 모두에서 일관된 경험을 동시에 볼 수 있도록 할 수 있습니다.

## 교차 애플리케이션 그룹화 {#cross-application}

기능 그룹은 애플리케이션 간 기능 관리를 지원합니다. 여러 애플리케이션의 관련 플래그를 함께 그룹화할 수 있습니다.
