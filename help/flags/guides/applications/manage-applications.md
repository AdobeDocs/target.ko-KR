# 애플리케이션 관리 {#manage-applications}

Flags의 **application**&#x200B;은(는) 기능 플래그로 제어할 서비스 또는 제품을 나타냅니다. 기능 플래그를 만들려면 먼저 하나 이상의 애플리케이션을 콘솔에 온보딩해야 합니다.

## 애플리케이션을 관리할 수 있는 사람 {#permissions}

**관리자** 역할을 가진 구성원만 응용 프로그램을 추가하거나 수정할 수 있습니다. 이 역할이 없는 경우 관리자에게 문의하십시오.

## 응용 프로그램 추가 {#add-application}

응용 프로그램을 추가하려면 [응용 프로그램 온보딩](onboard-your-application.md)에서 단계별 지침을 참조하십시오.
