# 기능 플래그 가져오기 {#import-feature-flags}

플래그를 사용하면 한 샌드박스(예: 샌드박스 1)에서 다른 샌드박스(예: 샌드박스 2)로 기능 플래그를 가져올 수 있습니다. 이렇게 하면 플래그 구성을 수동으로 다시 만들 필요가 없고 샌드박스 간 구성 드리프트가 발생할 위험이 줄어듭니다.

## 1단계: 대상 샌드박스 및 애플리케이션으로 이동 {#step-1}

**destination** 샌드박스(플래그를 가져올 샌드박스)의 콘솔에 로그인합니다. **기능 플래그 가져오기**&#x200B;를 클릭한 다음 플래그를 가져올 응용 프로그램을 응용 프로그램 드롭다운에서 선택합니다.

![기능 플래그 페이지의 기능 플래그 가져오기 단추](assets/import-feature-flags.png)

>[!IMPORTANT]
>
>현재 샌드박스와 선택한 응용 프로그램은 소스가 아닌 **대상**&#x200B;이어야 합니다. 예를 들어 샌드박스 1에서 샌드박스 2로 플래그를 가져오려면 샌드박스 2 콘솔에 로그인하고 샌드박스 2 애플리케이션을 선택합니다.

## 2단계: 가져오기 대화 상자 열기 {#step-2}

**기능 플래그 가져오기**&#x200B;를 선택하십시오. 사용 가능한 애플리케이션을 기반으로 미리 채워진 소스 샌드박스와 애플리케이션을 표시하는 대화 상자가 열립니다. 필요한 경우 대화 상자의 드롭다운에서 소스 샌드박스 및 애플리케이션을 변경할 수 있습니다.

## 3단계: 가져올 기능 플래그 선택 {#step-3}

소스 샌드박스의 기능 플래그 목록에서 가져올 플래그를 선택합니다. 한 번에 하나, 여러 개 또는 모든 플래그를 선택할 수 있습니다.

## 4단계: 가져올 기능 플래그 상태 선택 {#step-4}

드롭다운을 사용하여 기능 플래그를 가져오는 방법(**사용**, **사용 안 함** 또는 **현재 상태**)을 선택하십시오. 기본적으로 기능 플래그는 **사용 안 함** 상태로 가져옵니다.

## 중요 정보 {#important-notes}

기능 플래그를 가져올 때는 다음 사항에 유의하십시오.

* 동일한 키를 가진 기능 플래그가 대상 샌드박스에 이미 있는 경우 가져오지 않습니다.
