# 컨텍스트 속성 만들기 {#creating-your-context-attributes}

컨텍스트 속성은 사용자, 세션 또는 애플리케이션 컨텍스트(예: 구독 계층, 앱 버전 또는 지역)를 설명하는 사용자 지정 데이터 필드입니다. 컨텍스트 속성을 사용하여 기능 플래그에 대한 대상 기준을 정의합니다.

컨텍스트 그룹은 관련 컨텍스트 속성을 논리적 계층으로 구성합니다. 컨텍스트 그룹을 사용하여 기능 구성 중에 속성을 더 쉽게 찾고 관리할 수 있습니다.

| 용어 | 설명 | 다음에서 사용됨 |
| --- | --- | --- |
| **컨텍스트 특성** | 데이터 유형과 사전 정의된 선택적 값이 있는 명명된 필드. | 대상 기준, 기능 구성 |
| **컨텍스트 그룹** | 관련 컨텍스트 속성을 구성하는 범주입니다. | 피쳐를 생성할 때 컨텍스트 선택 |
| **UI 레이블** | 사용자 인터페이스에 표시된 표시 이름입니다. | 대상 기준 및 관리 페이지 |
| **ID** | 고유한 기술 식별자. | 애플리케이션 요청 |
| **미리 정의된 값** | 표시 레이블이 있는 허용된 값의 선택적 목록입니다. | 드롭다운 목록 및 대상 규칙 빌더 |

## 사전 요구 사항 {#prerequisites}

컨텍스트 속성 및 컨텍스트 그룹을 관리하기 전에 다음을 완료하십시오.

* 올바른 샌드박스에서 **플래그 콘솔**&#x200B;에 액세스할 수 있습니다.
* 컨텍스트 특성 및 컨텍스트 그룹을 만들고 관리하는 데 필요한 **관리자** 역할이 있습니다.

## 컨텍스트 속성으로 이동 {#navigate}

1. **플래그 콘솔**&#x200B;에 로그인합니다.
1. 왼쪽 탐색 메뉴의 **대시보드**&#x200B;에서 **컨텍스트 특성**&#x200B;을 선택합니다.

**컨텍스트 특성** 페이지에 있습니다. 이 페이지에서 컨텍스트 그룹 및 컨텍스트 속성을 만들 수 있습니다.

## 컨텍스트 그룹 만들기 {#create-group}

모든 컨텍스트 속성은 그룹에 속해야 합니다.

1. 왼쪽 탐색 메뉴의 **대시보드**&#x200B;에서 **컨텍스트 특성**&#x200B;을 선택합니다.
1. **내 컨텍스트 특성** 탭에서 **그룹 추가**&#x200B;를 선택합니다.
1. 필수 필드를 완료합니다.
1. **만들기**&#x200B;를 선택합니다.

### 그룹 필드 {#group-fields}

| 필드 | 설명 |
| --- | --- |
| **그룹 이름** | 그룹의 고유한 이름을 입력합니다. |
| **하위 그룹 만들기** | 계층을 구성할 상위 그룹을 선택하십시오. 속성을 만드는 동안 그룹을 만들 수도 있습니다. |

## 컨텍스트 속성 만들기 {#create-attribute}

1. **내 컨텍스트 특성** 탭에서 **새 컨텍스트 특성**&#x200B;을 선택합니다.
1. 필수 필드를 완료합니다.
1. **제출**&#x200B;을 선택합니다.

### 속성 필드 {#attribute-fields}

| 필드 | 설명 |
| --- | --- |
| **UI 레이블** | 대상 기준에 사용되는 표시 이름을 입력합니다. |
| **ID** | 응용 프로그램 요청에 사용되는 고유한 기술 식별자를 입력합니다. |
| **컨텍스트 그룹** | 그룹을 선택하거나 새 그룹 인라인을 만듭니다. |
| **데이터 형식** | 응용 프로그램에서 보내는 데이터와 일치하는 유형을 선택합니다. |

| 데이터 유형 | 예 |
| --- | --- |
| 문자열 | `gold` |
| 정수 | `42` |
| 부울 | `true` |
| 10진수 | `9.99` |
| 날짜 | `2026-07-17` |
| 버전 | `1.2.0` |

### 사전 정의된 값(선택 사항) {#predefined-values}

사전 정의된 값을 사용하여 속성을 고정된 값 세트로 제한합니다. **미리 정의된 값 추가**&#x200B;를 선택합니다.

| 필드 | 설명 |
| --- | --- |
| **UI 레이블** | 표시 이름을 입력합니다. |
| **값** | 애플리케이션에서 전송한 기술 값을 입력합니다. |

변경 내용을 적용하려면 **저장**&#x200B;을 선택하세요.

## 문제 해결 {#troubleshooting}

| 문제 | 해결 방법 |
| --- | --- |
| **만들기**&#x200B;이(가) 비활성화되었습니다. | **그룹 이름**&#x200B;을 모두 완료하고 **하위 그룹 만들기**&#x200B;를 완료하세요. |
| 속성을 제출할 수 없음 | **ID** 및 **컨텍스트 그룹**&#x200B;을 완료합니다. |
| 그룹이 드롭다운에 표시되지 않음 | 페이지를 새로 고치거나 속성 양식에서 인라인으로 그룹을 만듭니다. |
| 사전 정의된 값이 저장되지 않음 | 특성을 제출하기 전에 모달에서 **저장**&#x200B;을(를) 선택하십시오. |
