# 대상 규칙에서 컨텍스트 사용 {#context-in-audience-rules}

컨텍스트 변수는 런타임 시 클라이언트 애플리케이션에서 제공하는 값입니다. 이를 통해 사용자의 활성 언어, 장치 유형 또는 애플리케이션 상태와 같은 동적 세션 수준 정보를 기반으로 사용자를 타깃팅할 수 있습니다.

컨텍스트 변수는 웹 및 모바일 클라이언트와 관련이 있습니다.

## 컨텍스트 변수 작동 방식 {#how-context-works}

응용 프로그램에서 기능 플래그를 평가할 때 컨텍스트 변수를 플래그로 전달합니다. 콘솔에서 이러한 값을 확인하는 규칙을 정의하고, 플랫폼은 평가 시 이를 사용하여 사용자가 적격 상태인지 여부를 확인합니다.

## 컨텍스트 변수 유형 {#variable-types}

### 사전 정의된(목록) 유형 {#predefined-type}

언어 또는 국가 목록과 같이 고정된 허용 값 세트가 있는 컨텍스트 변수입니다.

사용 가능한 연산자: **Is**, **같지 않음**, **있음**, **있음**

### 정수 유형 {#integer-type}

숫자 값을 포함하는 컨텍스트 변수입니다.

사용 가능한 연산자: **다음보다 큼**, **다음보다 크거나 같음**, **같음**, **다음보다 작음**, **다음보다 작거나 같음**

### 문자열 유형 {#string-type}

자유 형식 텍스트 값을 포함하는 컨텍스트 변수입니다.

사용 가능한 연산자: **Is**, **같지 않음**

## 컨텍스트 변수 추가 {#adding-context-variable}

대상 규칙에 컨텍스트 변수를 추가하려면 다음 작업을 수행하십시오.

1. 콘솔에서 기능 플래그 또는 기능 그룹을 엽니다.
2. **대상자** 탭으로 이동합니다.
3. **Context**&#x200B;에서 새 조건을 추가합니다.
4. 컨텍스트 변수, 연산자 및 값을 선택합니다.

필요한 컨텍스트 변수가 목록에 표시되지 않으면 콘솔의 컨텍스트 변수 관리 섹션에서 셀프서비스 방식으로 새 변수를 작성할 수 있습니다.
