# 웹 서비스 {#web-services}

백엔드 서비스는 서버측 SDK을 사용하여 플래그와 통합됩니다. 기술 스택과 일치하는 SDK을 선택합니다.

## 사용 가능한 SDK {#available-sdks}

>[!NOTE]
>
>서버측 SDK 설명서는 현재 준비 중이며 곧 제공될 예정입니다. 조기 액세스 지침은 Adobe 담당자에게 문의하십시오.

## 기타 언어 {#other-languages}

특정 스택과의 통합에 대한 지침은 플래그 지원에 문의하십시오.
