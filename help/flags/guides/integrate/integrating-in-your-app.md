# 앱에 플래그 통합 {#integrate}

이 섹션에서는 응용 프로그램 유형별로 구성된 모든 플래그 클라이언트에 대한 통합 지침을 제공합니다.

## 통합 경로 선택 {#choose}

애플리케이션 유형과 일치하는 안내서를 선택합니다.

1. [시작 안내서](startup-guide.md) — 모든 통합 단계에 대한 높은 수준의 개요를 보려면 여기에서 시작하십시오.
1. [모바일 애플리케이션](mobile-applications.md) — 모바일 앱용 SDK 통합
1. [웹 응용 프로그램](web-applications.md) — 웹 속성에 대한 SDK 통합

## 애플리케이션 유형 개요 {#overview}

| 애플리케이션 유형 | 권장 통합 |
| --- | --- |
| **모바일 응용 프로그램** | AEP Mobile SDK — [Android](../sdk-releases/android/android-extension-integration-guide.md) 및 [iOS](../sdk-releases/ios/ios-extension-integration-guide.md) 안내서 참조 |
| **웹 응용 프로그램** | 웹 SDK - [웹 통합 안내서](../sdk-releases/web/web-extension-integration-guide.md) 참조 |
