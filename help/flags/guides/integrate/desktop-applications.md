# 데스크탑 애플리케이션 {#desktop-applications}

데스크탑 애플리케이션은 SDK을 통해 플래그와 통합됩니다. 데스크탑 스택과 관련된 통합 지침은 Adobe 담당자에게 문의하십시오.

>[!NOTE]
>
>Desktop SDK 설명서는 현재 준비 중이며 곧 제공될 예정입니다.

## 애플리케이션 ID {#app-id}

데스크톱 클라이언트는 **제품 코드 및 제품 버전**&#x200B;을 응용 프로그램 식별자로 사용할 수 있습니다.
