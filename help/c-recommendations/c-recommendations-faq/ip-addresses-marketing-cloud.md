오레곤 데이터 센터에 있는 권장 사항 피드 처리 서버에서 사용하는 IP 주소 목록으로서, Adobe 서버에서 시작되는 IP 주소를 허용하도록 방화벽을 구성하는 데 도움이 됩니다.

| CIDR 표기법 | 시작 IP | 종료 IP |
|---|---|---|
| 192.243.242.0/24 | 192.243.242.0 | 192.243.242.255 |
